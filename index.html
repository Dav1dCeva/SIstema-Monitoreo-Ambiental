Transcripción

SMAI es el Sistema de Monitoreo Ambiental Inteligente. Una plataforma web para capturar, monitorear y analizar datos ambientales en tiempo real.

Inicia sesión con tu correo y selecciona tu rol. Como administrador accedes al Dashboard, el centro de control del sistema.

Aquí ves un resumen general: total de lecturas, sensores activos, alertas vigentes. Cada sensor muestra su última lectura y estado. El gráfico semanal ayuda a detectar patrones.

En Umbrales configuras los límites de alerta para cada sensor. Presiona "+ Nuevo umbral", selecciona el parámetro, ingresa mínimo y máximo, y guarda. El sistema valida automáticamente las lecturas contra estos valores.

Consulta datos históricos con filtros por fecha, parámetro y ubicación. Exporta a PDF o Excel con un solo clic.

El módulo de Alertas registra cada vez que una lectura supera los umbrales. Puedes simular una alerta para probar el sistema.

Audiodescripción

El video comienza con el logo de SMAI, luego muestra la pantalla de inicio de sesión donde se ingresa un correo y se selecciona el rol Administrador. Tras iniciar sesión, se recorre el Dashboard con sus tarjetas de métricas, la grilla de sensores con estados y el gráfico de tendencia semanal.

A continuación se navega al módulo de Umbrales, se agrega un nuevo umbral para temperatura en Torre Norte con valores mínimos y máximos, y se guarda. Luego se muestra la Consulta de Datos con filtros aplicados y la exportación a PDF.

Finalmente se ingresa al módulo de Alertas, se simula una alerta y se observa cómo aparece en el historial. El video cierra con el logo de SMAI y el mensaje "datos accionables para un futuro sostenible".
